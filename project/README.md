# Vehicle Insurance Project

This project is organized around a workflow of data preparation, model training,
model selection, evaluation, inference, and fraud decision making.

## Directory Layout

- `skills/`: task-oriented skill folders for each stage of the workflow
- `fraud_oracle.csv`: placeholder dataset or output table for fraud decisions
- `notebook.ipynb`: main working notebook for the project

## Skills

- `preprocess-data`
- `train-models`
- `select-model`
- `evaluate-models`
- `run-inference`
- `fraud-decision-engine`
